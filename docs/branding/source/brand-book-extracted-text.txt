TYPOGRAPHY


BRANDON GROTESQUE IS THE ONLY HEADING TYPEFACE.
SOURCE SANS PRO IS THE ONLY BODY TYPEFACE.


PLEASE AVOID SUBSTITUTE, APPROXIMATE, OR ADD TO THIS SYSTEM.


Weight hierarchy:



Brandon Grotesque Bold                                 Brandon Grotesque Regular
Page titles, primary section headers                   Eyebrow labels, secondary UI text



Brandon Grotesque Medium                               Brandon Grotesque Light
Subheaders, card titles, navigation                    Large display text, hero headlines

TYPOGRAPHY


SIZING


The headline is set in Brandon Grotesque
Bold for both print and digital. All caps
with 25 tracking and 90% leading.

The subhead is set in Brandon Grotesque
Medium. All caps with 25 tracking. Start
at 1/3 headline size and scale up to a
maximum of 2/3 as desired or necessary
to fit the layout. With a lengthier headline,
start at 2/3 headline size and scale down
to a minimum of 1/3 as necessary.

The body copy is set in Source Sans
Pro Light (weight 300) for both print and
digital. 16px base with 0.04em tracking.

OUR VOICE


HOW F1R3FLY SHOULD SOUND:


•    Clear, technically accurate, no hype

•    Business-friendly, not academic

•    Confident and grounded—not "to-the-moon" Web3 marketing

•    Visionary, but anchored in real use cases



Approved tagline: "Concurrency for the people."

Preferred terms: verifiable infrastructure, audit-ready data, provable
systems, evidence-first

Avoid: investment language, unverified green claims, crypto hype,
aggressive competitor language, deep jargon on public-facing pages
