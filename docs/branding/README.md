# F1R3FLY Branding Resources

Start here:

- `brand-guide.md` - human-readable brand book equivalent
- `brand-architecture.yaml` - machine-readable endorsed-brand hierarchy

Machine-readable and implementation files:

- `tokens/` - color, typography, spacing, shadow, radius tokens
- `presets/` - CSS variables and Tailwind preset
- `assets/` - logos, gradients, and source/reference extracts
- `source/` - original source PDF and extracted text
